# 00-Hello

Smoke-test sample. Prints the Delphi-OBD package version. Confirms that
the runtime package compiles and can be referenced from an external
program.

## Build & run

```cmd
dcc32 -B Hello.dpr
Hello.exe
```

Expected output:

```
Delphi-OBD 2.0.0-alpha.0
(c) 2026 ERDesigns and Delphi-OBD contributors
https://github.com/erdesigns-eu/Delphi-OBD
```
