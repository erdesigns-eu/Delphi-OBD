# DoIPDiagnostics — ISO 13400 client

Demonstrates `TOBDDoIPClient` (the PLAN-spec'd "TOBDDoIP"). Covers
the full ISO 13400 payload-type surface: routing activation,
diagnostic message exchange, alive check, power mode, entity
status, and the vehicle-announcement event for UDP discovery.

```sh
dcc32 -B DoIPDiagnostics.dpr
DoIPDiagnostics
```
