# serial

Production-style serial transport setup with explicit baud rate, custom
header captions, and connection-status indicators.

- **Project:** `SerialDashboard.dpr`
- **Connection:** Serial (RS-232)
- **Complexity:** Intermediate

Set the COM port and baud rate (`38400`, `115200`, `9600`) in the form's
connection component before running. See [../README.md](../README.md) for
the full example catalog.
