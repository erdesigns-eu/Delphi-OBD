# tachograph

Digital tachograph / odometer example covering EU Gen1/Gen2 Smart
Tachographs and 8 international standards (EU, Korea, Russia, China,
Brazil, Japan, Australia). Auto-detects manufacturer (VDO, Stoneridge,
Denso) and supports 20+ commercial vehicle makes.

- **Project:** `TachographExample.dpr`
- **Feature:** Digital Tachograph
- **Complexity:** Advanced

Demonstrates: odometer read (total/trip/speed), 28-day driver activity,
workshop-card authentication, calibration and trip reset, VIN/event/fault
read, and DDD download.

See [../README.md](../README.md) for the full example catalog.
