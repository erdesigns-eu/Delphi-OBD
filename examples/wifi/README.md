# wifi

WiFi/TCP transport for network OBD-II adapters (typical default
`192.168.0.10:35000`). Reconnect-friendly setup suitable for unstable
networks.

- **Project:** `WifiDashboard.dpr`
- **Connection:** WiFi (TCP)
- **Complexity:** Intermediate

Set the adapter IP and port in the form's connection component before
running. See [../README.md](../README.md) for the full example catalog.
