# minimal

Smallest possible OBD-II application. Demonstrates connecting to an
ELM327-class adapter over a serial port, reading stored DTCs, and
clearing the MIL.

- **Project:** `MinimalDashboard.dpr`
- **Connection:** Serial (COM port, 38400 baud)
- **Complexity:** Beginner — read this first.

Edit the COM port in `MinimalDashboard.pas` (or the form) before running.
See [../README.md](../README.md) for the full example catalog.
