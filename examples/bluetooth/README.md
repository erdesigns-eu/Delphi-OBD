# bluetooth

Wireless dashboard using a paired Bluetooth OBD-II adapter (e.g. ELM327
clone). Demonstrates VCI status monitoring and connection-state
management for spotty wireless links.

- **Project:** `BluetoothDashboard.dpr`
- **Connection:** Bluetooth (RFCOMM)
- **Complexity:** Intermediate

Pair the adapter at the OS level first, then set the device name in the
form's connection component. See [../README.md](../README.md) for the
full example catalog.
