# ftdi

FTDI USB-serial adapter cable setup with guarded reconnect logic.
Targets diagnostic cables that expose an FTDI chip rather than a generic
serial port.

- **Project:** `FTDIDashboard.dpr`
- **Connection:** FTDI USB
- **Complexity:** Intermediate

Requires the FTDI D2XX driver to be installed. See
[../README.md](../README.md) for the full example catalog.
